🚀 AI Career Navigator

«Smart Career Roadmap & Skill Intelligence Platform powered by AI»

---

📖 Overview

AI Career Navigator is an intelligent web platform designed to help users discover, plan, and achieve their ideal tech career path.

Users can search for any skill (like Java, Python, UI/UX) and instantly get:

- 📍 Structured career roadmaps
- 🏭 Real-world industrial workflows
- 🧠 Role-based responsibilities
- 📊 Aptitude preparation guidance

The platform uses AI to generate real-time, industry-relevant content, making career planning more practical and actionable.

---

✨ Key Features

- 🔍 Smart Skill Search
Users can search any technology or role and get instant insights

- 🗺️ Dynamic Career Roadmaps
Beginner → Intermediate → Advanced structured path

- 🏭 Industrial Workflow Simulation
Real-world process of how the skill is used in companies

- 🧠 AI-Powered Content Generation
Uses Groq API to generate accurate, updated content

- 📊 Aptitude Preparation Section
Covers logical, quantitative, and interview-focused topics

- 🔐 Authentication System
Secure login & user management

- 📱 Responsive UI/UX
Optimized for both desktop and mobile devices

⚙️ Installation & Setup

1️⃣ Clone the Repository

git clone https://github.com/your-username/ai-career-navigator.git
cd ai-career-navigator

2️⃣ Create Virtual Environment

python -m venv venv
source venv/bin/activate     # Linux/Mac
venv\Scripts\activate        # Windows

3️⃣ Install Dependencies

pip install -r requirements.txt

4️⃣ Configure Environment Variables

Create a ".env" file and add:

GROQ_API_KEY=your_api_key_here

---

5️⃣ Run the Application

python app.py
